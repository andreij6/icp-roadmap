# Marketing — UI Kit

The public-facing landing page for Roadmap. Targets devs evaluating whether to host their project's roadmap on-chain.

## Sections
- `Nav.jsx` — slim, sticky, with II login
- `Hero.jsx` — headline + tagline + CTA, schematic visual
- `HowItWorks.jsx` — 3-step explainer (stake → vote → ship)
- `Showcase.jsx` — featured projects with vote counts
- `Footer.jsx` — minimal, monospaced
